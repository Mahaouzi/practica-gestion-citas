# Sistema de Gestión de Citas Médicas

## Descripción del proyecto

Este repositorio contiene la planificación y desarrollo inicial de un sistema de gestión de citas médicas para un hospital, realizado como parte de la asignatura de Ingeniería y gestión de proyectos software clínico.

El objetivo del proyecto es simular la organización y gestión de un proyecto software utilizando metodologías ágiles, en particular SCRUM, mediante el uso del sistema de gestión de tareas de GitHub.

El sistema propuesto permitirá gestionar las citas entre pacientes y médicos especialistas, facilitando la organización del trabajo del personal sanitario y administrativo.

---

## Funcionalidades principales

El sistema contempla las siguientes funcionalidades principales:

- Solicitud de citas médicas por parte de los pacientes.
- Validación de citas por el personal administrativo.
- Cancelación de citas médicas.
- Reprogramación de citas.
- Consulta del historial de citas del paciente.
- Visualización de estadísticas mediante un panel administrativo.
- Exportación de la agenda de los médicos en formato PDF.
- Filtros de búsqueda para localizar citas rápidamente.
- Pruebas de uso con personal clínico para validar el funcionamiento del sistema.

---

## Gestión del proyecto

La planificación del proyecto se ha realizado utilizando las herramientas de gestión de tareas de GitHub:

- Issues para representar tareas y funcionalidades.
- Labels para clasificar las tareas según prioridad, tipo o área funcional.
- Milestones para representar los distintos sprints del proyecto.
- Comentarios en las issues para simular la comunicación y coordinación entre los miembros del equipo.

Se han definido varios sprints para organizar el desarrollo del proyecto:

- Identificación de requisitos y diseño inicial del sistema
- Desarrollo de funcionalidades principales y gestión de incidencias
- Mejora del sistema y funcionalidades avanzadas
